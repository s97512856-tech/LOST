### قاعدة البيانات (SQLite)
- **users**: id, username, email, password (مشفرة), created_at
- **products**: id, name (EN+AR), description (EN+AR), price, category, file_url
- **orders**: id, user_id, total, status, payment_method, created_at
- **order_items**: id, order_id, product_id, quantity, price
- **carts**: id, user_id, product_id, quantity
- **reviews**: id, user_id, product_id, rating (1-5), comment
